<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/aw09/TestJavan.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```



<!-- USAGE EXAMPLES -->
## Usage
1. Create the database
    ```
    cat query.sql | sqlite3 database.db
    ```
2. Run 
    ```
    npm start
    ```
3. Open url
    ```
    http://localhost:3000
    ```
